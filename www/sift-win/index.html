Investing in SIFT Using Our Application

The SIFT Windows Client is an easy way to interact with SIFT. It wraps all of the contract functionality – such as dividends, transparency and purchasing SIFT during ICO, into a single application. Despite being super-easy to use it does require that you get a full local wallet up and running first.

Remember that you cannot invest in SIFT if you live in/are domiciled in the USA or any of its territories or if you are a USA citizen or permanent resident.

You can either just run Geth or run the Mist local wallet. There are plenty of guides already such as this one at medium or the official Ethereum readme. Whichever route you take it’s important to ensure that you have RPC enabled as this is how the SIFT Windows Client communicates with Ethereum.

Installation

You can always download the latest version of the client from our Github Repository. Get the latest version that isn’t marked as pre-release. Once you’ve installed it the first-time you’ll be prompted whenever a new version is available in-app. Download the MSI and double click it to install. The installation process will continue without prompting you.

Once the install has completed SIFT will start automatically. You can re-run it in the future from the application created in your start menu or from “C:\Program Files (x86)\Smart Investment Fund Token (SIFT)\sift-win.exe”.

When the application starts up a splash-screen will appear.

If this screen disappears within a few seconds you can move on to the “Investing in SIFT” section. It an error screen appears you may need to move onto the Configuration section.

If you ever see a screen inviting you to update it’s highly recommended you press Yes. This ensures all the latest features and functionality (such as voting on specific events and managing your funds after the ICO) are available to you.

Configuration and Error Checking

When you install the SIFT Windows Client it defaults to using RPC for communication with your Geth or Mist wallet on the default port of 8545. If you see the error splash screen rather than a screen inviting you to invest in SIFT you may need to configure either your Geth/Mist configuration or SIFT.

This screen indicates that SIFT Windows Client could not communicate with your local wallet. If you know your node uses an RPC port other than the default 8545 or it is not running on your local machine then you need to modify the SIFT Windows Client config. Start Notepad as an administrator and open c:\Program Files (x86)\Smart Investment Fund Token (SIFT)\sift-win.exe.config

You want to change the highlighted value to represent your RPC URL. If you haven’t setup RPC yet you can do this by ensuring the following values are included in your geth.exe parameters:
--rpc --rpcport "8545" --rpccorsdomain "*" --rpcapi "db,eth,net,web3,personal,miner" If you are running Mist then you need to prefix each parameter with –node: --node-rpc --node-rpcport "8545" --node-rpccorsdomain "*" --node-rpcapi "db,eth,net,web3,personal,miner"

You then need to restart either Mist or Geth and SIFT should connect. Please contact support if you have any further issues.

Investing in SIFT

Once the SIFT Windows Client starts it will show all the accounts in your local Ethereum wallet. If you haven’t got any accounts setup in your Mist/Geth wallet yet you’ll need to do that first and ensure there is enough money in at least one wallet to fund an investment in SIFT. If you don’t have a sufficient balance in your account to buy any SIFT, including transaction fees, you’ll be shown a screen asking you to top-up your account.

As soon as your balance is updated, or if you already have a balance, you’ll be shown the investor screen.

The top section shows you your account balance in both ether (or a smaller unit such as Wei, Szabo or Finney) and SIFT. In this example we’ve not yet purchased any SIFT but have a reasonable ether balance to buy SIFT with. The bottom section is a slider that lets you chose exactly how much SIFT you want to purchase. Move the slider left or right and you’ll be shown how much you are spending in ether on the left (0.06 ETH in this example) and how many SIFT you will purchase for that price on the right (6 SIFT here). When you are ready, press the Invest button. This button will not be enabled unless the ICO is open. The confirmation screen will then appear showing you how much you are spending, what the transaction fee is and which account you are sending it from. If you want to pay more or less of a transaction fee you can use the slider in the middle of the screen but the default values will suffice for most people. When you are happy enter your wallet password and press Invest.

Your transaction will then be processed. First, your transaction will be created – this should only take a few seconds. At this point it has been published to Ethereum and is waiting for a miner to broadcast it to the network.

The next stage is that your transaction will be confirmed. This is when a miner picks up your transaction and actually broadcasts it. This can take anywhere from seconds to minutes depending on how much of a transaction fee you paid and when you sent it.

Finally, your SIFT investment will be confirmed and the SIFT Windows Client balance will show your updated Ethereum and SIFT balances.

Congratulations! You are now an investor in SIFT.

Post ICO

Once the ICO has finished the SIFT Windows Client will be updated with many more features. Whilst you can always manage your SIFT holding in any Ethereum-wallet that lets you execute smart contracts we’ll always aim to provide the easiest user experience in our own client. If there’s demand we’ll also be releasing MacOS and Linux clients with limited functionality available on mobile devices. Some functionality that is planned for the future:

  • Sending SIFT to other accounts
  • Authorising others to send SIFT on your behalf
  • Extracting your dividends on pay day
  • Monitoring the value of the fund
  • Viewing transparency information supplied by SIFT

If you have any other ideas of how we can enhance the SIFT investor experience we’d love to hear it.
